# Програмний комплекс для аналізу відеоконтенту

Цей проект є вебмікросервісом для автоматизованого завантаження, сегментації та семантичного аналізу відеоконтенту. Система спроектована для обробки лекцій, подкастів та інших інформаційних відео, виконуючи їх розбиття на логічні сцени з транскрибацією та виділенням ключових тем.

## Основні можливості

- **Завантаження:** Підтримка завантаження локальних файлів та посилань з Google Drive.
- **Сегментація:** Автоматичне визначення монтажних переходів за допомогою нейромережі TransNet V2 з подальшою фільтрацією результатів (Gaussian Smoothing).
- **Аналіз звуку:** Транскрибація мовлення за допомогою моделі Whisper.
- **Семантичний аналіз:** Визначення ключових тем відео та окремих сцен з використанням бібліотек `langdetect` та `stop-words`.
- **Експорт:** Можливість отримання результатів у форматах JSON та TXT.

## Технічний стек

- **Backend:** FastAPI, Python 3.14
- **Медіа-обробка:** FFmpeg
- **Моделі:** TransNet V2, OpenAI Whisper
- **Зберігання:** SQLite3 (SQLAlchemy)

## Архітектура програмного конвеєра

Процес обробки відео організований у вигляді асинхронного програмного конвеєра:
1. **Ініціалізація:** Валідація вхідних даних та генерація унікального `job_id`.
2. **Зондування:** Перевірка медіаданих та необхідності перекодування.
3. **Обробка:** Сегментація, транскрибація та аналіз метаданих.
4. **Завершення:** Агрегація результатів та фільтрація стоп-слів.

## Налаштування та запуск

### Вимоги
- Встановлений [FFmpeg](https://ffmpeg.org/) у системі.
- Python 3.14+

### Інсталяція залежностей
bash
pip install -r requirements.txt
Запуск сервера
Для запуску локального сервера використайте скрипт:

Bash
python run_local.py
Після запуску інтерфейс буде доступний за замовчуванням на порту 8000.

Обробка винятків
Система передбачає стійкість до помилок:

Некоректні вхідні дані: HTTP 422 у разі валідаційних помилок.

Помилки доступу/формату: Перехоплення винятків під час роботи програмного конвеєра з автоматичним переведенням задачі у статус failed та записом причини збою.
